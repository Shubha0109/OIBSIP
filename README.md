# 🌐 OIBSIP - Web Development Internship Projects

This repository contains the projects completed during my Web Development Internship at Oasis Infobyte. These projects helped me understand the fundamentals of frontend development and improve my practical skills.

---

## 🚀 Projects Included

### 🔹 1. Landing Page
A simple coffee shop landing page created using HTML and CSS. It includes sections like header, about, menu, and footer.

### 🔹 2. Portfolio Website
A personal portfolio website to showcase my details, skills, and projects. Built using HTML and CSS.

### 🔹 3. Temperature Converter
A web application that converts temperature between Celsius and Fahrenheit using JavaScript.

### 🔹 4. To-Do List
A dynamic web application where users can add, delete, and mark tasks as completed. Built using HTML, CSS, and JavaScript.

### 🔹 5. Login Authentication System
A login system with registration and login functionality. It uses JavaScript and localStorage to store and validate user credentials. Includes features like password toggle and input validation.
