## Overview
This is a professional To-Do App created for the Oasis Infobyte Web Development & Design Internship (OIB-SIP).  
It demonstrates HTML, CSS, and JavaScript skills, along with DOM manipulation and LocalStorage.

## Features
- Add tasks with a single input box
- Edit existing tasks
- Delete tasks
- Mark tasks as Completed or Pending
- Separate sections for Pending & Completed tasks
- Tasks are saved in browser LocalStorage and persist after page refresh
- Attractive UI with color-coded tasks and hover effects
- Fully interactive demo available in video

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage (Browser)
